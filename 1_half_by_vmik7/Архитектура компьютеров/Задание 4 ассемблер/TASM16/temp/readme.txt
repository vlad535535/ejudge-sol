3. Жёстко следим за типами данных:
В стек можно помещать только word.
AX - это word, состоящий из AH и AL, которые в свою очередь являются byte.

4. Прототип процедуры (поместить в конец сегмента кода)
print proc			;print - имя процедуры (произвольное)
	push bp			;(*)сохраняем содержимое bp для последующего восстановлени,
					;т.к. в ходе процедуры мы будем его использовать
	mov bp,sp		;помещаем в bp адрес последней записи стека из sp
	mov ax,[bp+4]	;первый аргумент вытаскиваем из стека следующим образом:
					;по адресу bp хранится значение из пункта (*) размером в 2 байта (1 слово)
					;следующий элемент стека (размер - также 1 байт) - это возвращаемое значение функции (адрес bp+2)
					;ну и первый аргумент, который мы положили в стек перед вызовом функции: адрес bp+4
					
	;... что-то делаем
	
	pop bp			;восстанавливаем bp
	ret				;обязательная строчка, возвращаемся к месту вызова функции
print endp 

//подробнее: http://asmworld.ru/uchebnyj-kurs/025-peredacha-parametrov-cherez-stek/

5. Процедуру вызывать так: call print
print - имя процедуры.
